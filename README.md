## ğŸš€ Getting Started

To get started with developing this application locally, follow these steps:

### Prerequisites

Make sure you have the following installed:
- [Node.js](https://nodejs.org/en/) (v18 or later is recommended)
- A package manager like `npm`, `yarn`, or `pnpm`. This guide will use `npm`.

### Running in VS Code

1.  **Open the Terminal:**
    Open the project folder in Visual Studio Code. Then, open the integrated terminal by pressing ``Ctrl+` `` (or `Cmd+\`` on macOS).

2.  **Install Dependencies:**
    In the terminal, run the following command to install all the required packages for the project:
    ```bash
    npm install
    ```

3.  **Run the Development Server:**
    Once the dependencies are installed, start the Next.js development server:
    ```bash
    npm run dev
    ```

4.  **View the Application:**
    The application will now be running. You can view it by opening your web browser and navigating to:
    [http://localhost:3000](http://localhost:3000)

The main entry point of the app is `src/app/page.tsx`, where you can explore and modify the core functionality.
